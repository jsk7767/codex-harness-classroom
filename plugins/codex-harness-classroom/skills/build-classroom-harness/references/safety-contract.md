# 근거 안전 계약

- Obsidian vault는 사용자가 지정한 회사 지식창고다.
- 메뉴, 가격, 할인, 영업시간, 주차, 후기, 고객 발언, 주소, 연락처 및 사업 관련 사실은 사용자 제공 vault 루트 기준 상대 경로를 근거로 제시해야 한다.
- 근거 문법은 `[근거: notes/store.md]` 하나만 사용한다.
- 근거가 없으면 줄을 `[확인 필요]` 또는 `> [확인 필요]`로 시작하고 홍보 문구로 확정하지 않는다.
- 제목과 확인 필요 줄을 제외한 모든 게시 가능 내용 줄은 근거가 없으면 FAIL이다.
- Windows PowerShell에서 한글 파일을 읽을 때 UTF-8을 명시한다. 글자가 깨지면 추측하지 말고 중단한 뒤 UTF-8로 다시 읽는다.
- Fact Checker는 vault를 읽기만 한다. Content Writer는 Fact Checker가 승인한 사실만 쓴다. Quality Checker는 원문과 독립 대조한다.
- 독립적인 읽기 전용 감사만 병렬로 실행한다. 같은 문서를 쓰는 담당 직원은 반드시 직렬로 실행한다.
